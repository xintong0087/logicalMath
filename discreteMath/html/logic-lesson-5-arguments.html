Lesson 5

Valid Arguments & Fallacies

Name:   Date:

Arguments: premises and conclusion

An argument is not a fight — it is a structured list:

  • Premises — statements you assume for the moment.
  • Conclusion — what you claim follows from them.
Example argument
  1. All mammals breathe air. (premise)
  2. Whales are mammals. (premise)
  3. Therefore whales breathe air. (conclusion)

Valid vs sound

  • Valid: IF the premises were true, the conclusion must be true. Structure is correct.
  • Sound: Valid and every premise is actually true in the real world.

A valid argument can still have a false conclusion if a premise is false. Validity is about the pattern, not whether you agree with the facts.

Valid patterns to recognize

Modus ponens

p → q · p · therefore q

If it rains, the field is muddy. It rains. ∴ The field is muddy. ✓

Modus tollens

p → q · ¬q · therefore ¬p

If the alarm is armed, the light is red. The light is not red. ∴ The alarm is not armed. ✓

Invalid patterns — classic traps

Affirming the consequent ✗

From p → q and q, concluding p.

“If you study hard, you pass. You passed. So you studied hard.” — Maybe, but not forced by logic.

Denying the antecedent ✗

From p → q and ¬p, concluding ¬q.

“If it rains, the game is canceled. It didn’t rain. So the game wasn’t canceled.” — Wrong; they might cancel for wind.

More fallacies (everyday names)

  • False dilemma: “Either you’re with us or against us” when middle options exist.
  • Hasty generalization: “My friend got sick after sushi, so all sushi is unsafe.”
  • Ad hominem: Attacking the person instead of the argument.

De Morgan’s laws (simplifying negations)

  • ¬(p ∧ q) ≡ (¬p ∨ ¬q) — “Not both” = “at least one is false.”
  • ¬(p ∨ q) ≡ (¬p ∧ ¬q) — “Neither” = “not this and not that.”
Worked example

“It is not true that the store is open and I have cash.”

Let p = store open, q = I have cash. Original: ¬(p ∧ q). De Morgan: ¬p ∨ ¬q = “The store is closed OR I don’t have cash (or both).”

5.1 Warm-up

Label premises and conclusion:

“Soccer is a sport. All sports require practice. Therefore soccer requires practice.”

5.2 Warm-up

Rewrite using De Morgan: ¬(p ∨ q). Then put it in plain English if p = “I like math” and q = “I like art.”

Symbolic + English
5.3 Standard

Valid or invalid? Name the pattern if you can.

  1. If n is divisible by 10, then n is divisible by 5. n is divisible by 10. So n is divisible by 5.
  2. If you wear a helmet, you are safe. You are safe. So you wore a helmet.
  3. If it is Friday, the cafeteria serves pizza. It is not Friday. So no pizza today.
Valid/invalid + pattern name
5.4 Standard

Fix this invalid argument by changing only the conclusion so the argument becomes valid (premises stay the same):

P1: If a bird is a penguin, it cannot fly.
P2: Tweety is not a penguin.
Bad conclusion: Tweety can fly.

New valid conclusion + why
5.5 Standard

Build a truth table for (p → q) ∧ p. When the last column is true, what must be true about q? Connect to modus ponens.

5.6 Challenge

Write a valid argument with two premises and one conclusion about a topic you know (sport, game, school rule). Then write a invalid argument on the same topic that looks similar but commits affirming the consequent or denying the antecedent.

Valid argument / invalid argument
5.7 Challenge
