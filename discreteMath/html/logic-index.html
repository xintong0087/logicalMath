Logic for Beginners

Discrete-style reasoning for 9th grade · six lessons · open one file per session

Logic is the study of good reasoning. Each lesson below is a separate HTML file with teaching notes and practice problems from Warm-up through Standard to Challenge. Work through them in order.

Lessons
  1. Statements and truth
    What counts as a proposition, truth values, open vs closed sentences
  2. NOT, AND, OR
    Combining statements, inclusive OR, double negation
  3. Truth tables
    Building tables step by step, negating compounds
  4. If…then & biconditionals
    Conditionals, converse / inverse / contrapositive, iff
  5. Valid arguments & fallacies
    Premises, validity, modus ponens/tollens, De Morgan
  6. Puzzles & review
    Knights & knaves, real-world reasoning, mixed review
Suggested pace: one lesson per session (~40–50 minutes): read the notes together, try warm-ups aloud, then let the student work standard and challenge problems in the blank boxes (type or print).
