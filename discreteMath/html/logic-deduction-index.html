Deduction Problems

Hard, step-by-step practice · one set per lesson topic · no shortcuts

These problem sets sit beside the six regular lessons. Each problem is designed to reward patience: you must show every case, every row, or every branch of reasoning. Guessing or pattern-matching without justification will not earn credit.

Rules for students: (1) Write intermediate steps — do not jump to the final answer. (2) If a problem asks for “all” assignments or “every” row, incomplete work is wrong even if the final guess is right. (3) Use the blank boxes; print if you prefer pencil and paper.
Problem sets by topic
  1. Statements and truth
    Borderline classification, open sentences, fixed-domain reasoning
  2. NOT, AND, OR
    Exhaustive assignments, nested compounds, equivalence checks
  3. Truth tables
    Full 8-row tables, sub-columns, tautology and equivalence proofs
  4. If…then & biconditionals
    Contrapositive chains, nested conditionals, biconditional puzzles
  5. Valid arguments & fallacies
    Multi-step validity, modus ponens chains, De Morgan + logic
  6. Puzzles & synthesis
    Knights & knaves at scale, mixed review, full case analysis

Suggested use: assign one problem set after the matching lesson, or spread them across review weeks. Expect 60–90 minutes per set for careful work.
