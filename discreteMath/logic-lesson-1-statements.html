Lesson 1

Statements and Truth

Name:   Date:

What is logic?

Logic helps you decide whether a chain of reasons actually supports a conclusion. Scientists, lawyers, programmers, and mathematicians all use the same basic toolkit — we start with the simplest piece: a statement.

Statements (propositions)

A statement (also called a proposition) is a declarative sentence that has exactly one truth value: it is either true or false, not both, and not “sort of.”

Statements — each is clearly T or F
  • “Melbourne is in Australia.” → True
  • “The sum of 2 and 3 is 6.” → False
  • “All prime numbers greater than 2 are odd.” → True
  • “There are exactly seven planets in our solar system.” → depends on definition, but astronomers treat this as a fixed claim that is false (eight planets since Pluto’s reclassification)

Not statements — common categories

  • Questions: “Is homework due tomorrow?” — asks for information; no truth value.
  • Commands: “Show your work.” — tells someone to act.
  • Exclamations: “That goal was incredible!” — expresses feeling; not objectively T/F.
  • Pure opinions (without a clear standard): “Strawberry ice cream is the best flavor.” — debatable, not a fixed fact in logic class.
  • Paradoxes (skip for now): “This sentence is false.” — breaks the true/false rule.
Borderline cases: “The test was hard” sounds like an opinion, but if we agree on a rule (“hard = more than half the class scored below 60%”), it could become a statement. In logic, we need a clear standard before we call something true or false.

Labeling statements with letters

We use lowercase letters as nicknames:

  • Let p = “It is raining.”
  • Let q = “The ground is wet.”

On a dry sunny day, p is false and q might also be false. After a storm, both might be true. The letters stay the same; the world decides T or F.

Open sentences vs statements

An open sentence contains a variable and is not a statement until the variable is fixed.

Open vs closed

Open: “x + 5 = 12” — true when x = 7, false when x = 0.

Closed (statement): “7 + 5 = 12” → True.

Open: “Student n passed the exam.” — depends on which student.

Closed: “Maria passed the exam.” → one fixed T or F.

Key idea Algebra and geometry use statements all the time. “If x = 4, then x2 = 16” is a true conditional. Logic gives the grammar for linking such sentences correctly.

Atomic vs compound (preview)

A single statement like “Snow is white” is atomic (one piece). “Snow is white and grass is green” is compound — built from smaller parts. Lesson 2 covers the builders (AND, OR, NOT).

  2. Close the window.
  3. Every square has four sides.
  4. Who invented the telephone?
