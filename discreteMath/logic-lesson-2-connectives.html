Lesson 2

NOT, AND, OR

Name:   Date:

Compound statements

Simple statements can be combined with logical connectives. Think of them as glue between ideas. We use symbols so long sentences stay readable.

NOT · ¬p
“It is not the case that p.”

Flips T ↔ F. Also written “~p” in some books.

AND · p ∧ q
“p and q.”

True only when both parts are true.

OR · p ∨ q
“p or q” (inclusive)

True when at least one part is true (including both).

NOT in detail

If p = “The door is locked” is true, then ¬p = “The door is not locked” is false.

Double negation

Negating twice returns to the original truth: ¬(¬p) has the same truth value as p.

“It is not true that I did not finish” → means I did finish (clumsy English, clear logic).

AND in detail

AND is strict. One false part sinks the whole compound.

  • “I have a ticket and I have ID” → need both to enter.
  • True ∧ False → False
  • False ∧ False → False

OR in detail — inclusive vs everyday “pick one”

In logic, OR is inclusive: both parts can be true and OR is still true.

Inclusive OR

“You may have soup or salad” at a buffet often means you can take both. Logical OR matches that generous reading.

Exclusive OR (“exactly one, not both”) is a different connective — we write XOR or “p ⊕ q” in advanced courses. Unless a problem says “exactly one,” assume inclusive OR.

Parentheses matter

Like arithmetic, grouping matters: ( p ∧ q ) ∨ r is not the same as p ∧ ( q ∨ r ). Always write parentheses when mixing connectives.

Worked example

Let p = “I finished homework.” (T) · q = “I studied for the quiz.” (F) · r = “I cleaned my room.” (T)

  • ¬q → True
  • p ∧ q → False
  • p ∨ q → True
  • (p ∧ q) ∨ r → (F) ∨ T → True
  • p ∧ (q ∨ r) → T ∧ (T) → True
Memory trick AND is picky (all T). OR is generous (one T enough). NOT is a light switch.

Practice — Lesson 2

2.1 Warm-up

Let p be true and q be false. Find the truth value:

  1. ¬p
  2. ¬q
  3. p ∧ q
  4. p ∨ q
T or F for each
2.2 Warm-up

Translate into symbols. Define p and q first.

  1. The store is open and I have money.
  2. It is not raining.
  3. I will take the bus or I will walk.
Definitions + symbolic form
2.3 Standard

p = T, q = T, r = F. Evaluate:

  1. ¬(p ∨ r)
  2. (p ∧ r) ∨ q
  3. ¬p ∨ (q ∧ r)
Show intermediate steps
2.4 Standard

Write each English sentence in logic symbols (define letters).

  1. I am not tired and I am not hungry.
  2. Either the red team or the blue team wins (or both tie — use inclusive OR).
  3. It is false that the library is closed and the café is closed.
Symbol versions
2.5 Standard

For each pair, explain in one sentence whether they mean the same thing:

  1. “You need a password and a username.” vs “You need a username and a password.”
  2. “Pick coffee or tea (choose only one).” vs logical OR without the “only one” rule.
Same or different? Why?
2.6 Challenge

p, q, r are all unknown. Find one assignment of T/F to p, q, r that makes the whole compound true, and one that makes it false:

(p ∨ ¬q) ∧ (q ∨ r) ∧ ¬(p ∧ r)

True assignment / false assignment + brief check
2.7 Challenge

Write a real-life rule (school, sport, or game) using at least two connectives. Give definitions for your letters and explain when the rule passes vs fails.

Rule + analysis
