Lesson 3

Truth Tables

Name:   Date:

Why truth tables?

A truth table lists every way the parts of a compound statement can be true or false, and shows the result for the whole expression. With two basic statements p and q, there are four rows: TT, TF, FT, FF.

Recipe for building a table

  1. Count variables: n variables → 2n rows.
  2. Write columns for each variable (often p changes slowest, q fastest — either pattern is fine if you are consistent).
  3. Add columns for sub-expressions from the inside out (parentheses first).
  4. The final column is the whole statement.

English: “Not both” is false only when p and q are simultaneously true.

Tautology and contradiction (names worth knowing)

  • Tautology: always true (final column all T). Example: p ∨ ¬p.
  • Contradiction: always false (final column all F). Example: p ∧ ¬p.
Three variables: With p, q, r you need 8 rows. Challenge problems may use three — work systematically so you do not skip a case.
